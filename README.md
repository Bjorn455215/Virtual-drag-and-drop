# Virtual-drag-and-drop
利用OpenCV 及 Mediapipe 實現簡易手勢滑動視窗的操作

## 專案簡述
本專案利用 Python 與 OpenCV 打造一個仿《鋼鐵人》風格的擴增實境（AR）互動介面。透過 MediaPipe 進行高精度的手勢追蹤，實現了直覺的虛擬視窗抓取、移動、以及動態生成功能。

## 核心功能
* **三指捏合偵測** ：整合大拇指、食指與中指的空間座標運算，計算中心控制點，實現比雙指捏合更穩定、更符合人體工學的抓取感。
* **智慧型視窗層級管理** ：唯一抓取邏輯：利用倒序遍歷與中斷機制，確保在視窗完全重疊時，僅會觸發最上層物件，杜絕「一次抓起一整疊」的邏輯衝突。
* **動態置頂** ：當物件被選中時，自動將其移動至清單末端（最高渲染層級），確保移動過程中不會被其他視窗遮擋。
* **動態科技 UI 渲染**：半透明質感：採用雙層畫布疊加技術，營造虛擬影格投影在空氣中的透明質感。
* **科技風格**：包含動態掃描線效果、霓虹青配色，以及自動編號的數據節點標籤。
* **手勢新增**：監控食指與大拇指之張開距離（> 180 像素），模擬從虛空中「拉出」新視窗的直覺體驗，並內建冷卻時間防止重複觸發。
* **物件導向架構**：視窗行為由 DragRect 類別封裝，可擴充視窗的尺寸、內容與功能。

## 環境需求
*   **作業系統**：本專案利用Windows
*   **Python 版本**：使用 **3.9**，因為安裝Mediapipe只能搭配**3.10以下的版本**
*   **操作環境**: 使用VS Code
*   **硬體**：具備網路攝影機 (Webcam)

## 操作方式
本系統完全採用無接觸手勢進行互動，攝影機視野需光線充足。

| 手勢 (Gesture) | 視覺示意 | 功能描述 (Function) | 技術實現 |
| :--- | :---: | :--- | :--- |
| **三指捏合** | 👌 | **抓取與移動視窗** | 偵測大拇指、食指與中指中心點，鎖定並拖拽最上層物件。 |
| **L型張開** | 🖐️ | **新增數據視窗** | 當指尖距離大於 180 像素時，於手心位置產出新節點。 |
| **視窗重疊** | 🗂️ | **智慧置頂** | 選中的視窗會自動切換至最前端 (Bring to Front)，防止被遮擋。 |
| **CTRL + C 鍵** | ⌨️ | **關閉系統** | 在終端輸入安全釋放攝影機資源並退出程式環境。 |

> **抓取技巧**：當多個視窗重疊時，系統會自動優先判斷最前方的視窗。若要移動下層視窗，請先將上層視窗移開。
